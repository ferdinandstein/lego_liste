# Neue Sets hinzufügen

1. `set_list.py` aktualisieren und die Set Ids nach dem Shema eintragen.
    Wenn das kein Legoset ist, dann in `create_database.py` eintragen in der Statistik Funktion
1. `import_set_info.py` ausführen.
1. `import_parts.py` ausführen.
1. `create_database.py` ausführen.
1. Das ganze committen.
1. Das ganze pushen.

# Wenn sich die Statistiken nicht ändern, dann liegt das daran, das der Browser die alten Daten noch im Cache hat. Einmal Browser schließen.
